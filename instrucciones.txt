Inicializar proyecto
    npm init -y

Inicializar repositorio
    git init

Crear Archivos a nivel raiz
    .gitignore  => sacar contenido de gitignore.io(https://www.toptal.com/developers/gitignore/)
        (node,windows,linux,mac)
    .editorconfig
        Instalar extension en VSC editorConfig for VSC
    .eslintrc.json
    index.js


Instalar Dependencias
  npm i nodemon eslint eslint-config-prettier eslint-plugin-prettier prettier -D

  npm i express

  npm i @faker-js/faker => para trabajar con datos falsos

  npm i @hapi/boom   => manejo de errores con boom

  npm i joi   => validación de datos con Joi (esquemas)

  npm i cors  => Acceso de dominios a la api
